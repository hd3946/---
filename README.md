# Project_chat-bot  (과목: 오픈소스)
-  Dialogflow를 이용한 챗봇 만들기

#### 프로젝트 목표
 * 기본 chat 동작 (ex 인사, 시간 스몰톡)
 * 데이터베이스 활용 시나리오 (DB사진업로드 후 사진불러오기)
 * Wep API를 활용 시나리오 (Twitter api를 이용해 특정 연예인 글 가져오기)
